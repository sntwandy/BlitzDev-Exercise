# BlitzDev Technical Test

A project using React.js, TailwindCSS and Storybook.

## Requirements to run the project 🚀

You need to have Node V14 or higher.

#### To install the dependencies:

```bash
  yarn install
```

or

```bash
  npm install
```

#### To run the project:

```bash
  yarn dev
```

or

```bash
  npm run dev
```

#### To run storybook:

```bash
  yarn storybook
```

or

```bash
  npm run storybook
```
